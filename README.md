## Stack

Plain HTML, CSS, and JavaScript. No build step. No framework. Hosted on GitHub Pages.

Pages:

- `index.html` — Home (Arabic primary, EN toggle)
- `events.html` — Events & PR
- `brand.html` — Brand Identity
- `community.html` — Community Building & Influencer Management

Shared:

- `styles.css`
- `script.js`
- `assets/` — images, photos, source materials

## Local preview

Open `index.html` in a browser. Or run a local server:

```
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
